Namespace: Icons

Icons

A collection of reusable SVG icons for the application.

Each property is a string containing inline SVG markup, which can be directly inserted into the DOM (e.g., via innerHTML).

Properties:
Name Type Description
edit string

Pencil/edit icon SVG.
delete string

Trash/delete icon SVG.
Source:
  • utils/icons.js, line 1

Example

// Insert edit icon into a button
const button = document.createElement("button");
button.innerHTML = Icons.edit;
document.body.appendChild(button);
